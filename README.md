# 📊 SkillFlow Dashboard — IT-Park Test #5

Ushbu loyiha mening **IT-Park** o'quv markazidagi HTML va CSS bo'yicha yakuniy va eng murakkab testlarimdan biridir. Loyiha davomida real interfeys (Dashboard) yaratish, elementlarni murakkab joylashtirish va foydalanuvchi tajribasini (UX) hisobga olish ko'nikmalarimni namoyish etdim.

## 🏆 Erishilgan natija
Ushbu bosqichga kelib, HTML/CSS bo'yicha bilimlarim sezilarli darajada oshdi. Loyiha to'liq **Layout design**, **Component building** va **Visual hierarchy** qoidalari asosida qurilgan.

## 🛠 Texnologiyalar
* **HTML5** — Semantik dashboard strukturasi.
* **CSS3 (Advanced)** — Murakkab dizayn va vizual elementlar uchun:
    * **Flexbox & Grid** — Navigatsiya va kartalar joylashuvi uchun.
    * **Progress Bars** — Kurslar tugatilish foizini ko'rsatuvchi dinamik barlar.
    * **Sidebar & Header** — Foydalanuvchi boshqaruv paneli dizayni.
    * **Typography & Spacing** — `line-height`, `letter-spacing` va `padding/margin` ustida aniq ishlangan.

## ✨ Loyiha tarkibi (Project Features)

* **Sidebar Navigation:** Boshqaruv menyusi (Dashboard, Courses, Progress, etc.).
* **Welcome Section:** Foydalanuvchini tabriklovchi va umumiy progressni ko'rsatuvchi qism.
* **Quick Stats:** O'tilgan darslar, soatlar va sertifikatlar soni ko'rsatilgan statistik bloklar.
